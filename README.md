# Personal Portfolio

My personal portfolio site. One page that introduces who I am, what I work with, the projects I have built, and how to reach me. It ties together the rest of my work in one place.

## What is on it

- A short introduction and summary of my background
- The tools and technologies I use day to day
- Project cards linking to live demos and source code
- A timeline of my work experience
- Contact details, plus links to my GitHub and LinkedIn
- A button to download my CV

## Built with

- React (with Vite)
- Plain CSS, with a dark theme and a responsive layout that works on phones and laptops

## Running it locally

```bash
npm install
npm run dev
```

Then open the local address it prints (usually http://localhost:5173).

To build the production version:

```bash
npm run build
```

The finished site lands in the `dist` folder, ready to deploy on Netlify, Vercel, or GitHub Pages.

## A couple of things to set before deploying

- In `src/App.jsx`, swap the placeholder `#` links on the project cards for your live demo and repository URLs
- Drop your CV into `public/` so the download button works
